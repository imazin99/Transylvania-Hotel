**🏨 Transylvania Hotel Website**                                                                   
A simple and visually engaging web project inspired by Hotel Transylvania.
**🎯 Purpose**                                                                              
This project was created for learning and practice purposes.                                  
It focuses on applying core web development concepts such as:                                
Structuring web pages using HTML                                                                    
Styling and layout design using CSS                                                                  
Creating simple interactive UI elements                                                    
**✨ Features**                                                                              
🎬 Information about Hotel Transylvania and its sequel                                              
🧭 Clean navigation bar                                                                      
🌌 Cinematic background with blur effect                                                  
💡 Hover animations and visual effects                                                    
**🛠️ Technologies Used**                                                                                  
HTML5                                                                                      
CSS3
